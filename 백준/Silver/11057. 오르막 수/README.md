### 아이디어

- 오르막 숫자가 되기 위해서는 앞의 숫자보다 큰 숫자가 이어 나온다 
- 즉, 현재 숫자보다 앞에는 더 작거나 같은 숫자만 올 수 있다
- dp[i][j] : i자리 수에서 끝에 j가 올 수 있는 경우의 수
- 따라서 점화식은 dp[i][j] +=dp[i-1][j보다 작거나 같은 숫자]

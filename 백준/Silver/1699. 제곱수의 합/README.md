### 아이디어

- dp[i] : i를 만들 수 있는 제곱수의 합 경우의 수
- i를 만들 수 있는 제곱수의 합을 구하기 위해 될 수 있는 제곱수는 i의 제곱근 까지이다.
- 즉, 점화식은 1부터 i의 제곱근까지 범위에 대해 dp[i] = min(dp[i],dp[i-j * j]+1)이다.

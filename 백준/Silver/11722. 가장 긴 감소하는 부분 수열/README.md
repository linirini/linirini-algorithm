### 아이디어

- dp[i] : i번째까지 감소하는 수열의 가장 긴 길이의 값
- i번째 이전까지의 입력값(j번째 입력값)들 중 현재의 값보다 크면서 그 dp[j]+1값이 현재 dp[i]의 값보다 클 때 갱신
- 따라서 점화식은 if (A[j] > A[i] && dp[i] < dp[j] + 1) dp[i] = dp[j] + 1;

메모리: 2020 KB, 시간: 12 ms

### 분류

### 아이디어
- 번호 순서대로 본인의 위치까지 swap 해오기
- swap 한 횟수 카운트
- 예시
  - 3 2 4 1
    - swap(4,1) -> 3 2 1 4
    - swap(2,1) -> 3 1 2 4
    - swap(3,1) -> 1 3 2 4
    - swap(3,2) -> 1 2 3 4

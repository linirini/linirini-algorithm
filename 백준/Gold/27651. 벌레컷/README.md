### 아이디어

- 자료형 주의...........
- 머리-가슴-배
- 머리 < 배 && 배 < 가슴
- 배의 길이 마다, 가능한 머리&가슴 영역을 카운트하자
  - 구간합 활용
  - 배의 길이를 변경해가며 머리<배<가슴을 만족하는 index를 찾는다.
  - index는 이분 탐색 활용 -> 가장 큰 index 찾기
  - 찾은 index == 경우의 수

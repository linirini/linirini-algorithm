### 아이디어
- **연속**된 전구 상태 뒤집기 + **무조건 하나는 뒤집어야 한다** + 현재 상태에서 **얼마나 늘어날 수 있을까**
- 앞선 전구 모두를 뒤집고, 현재 전구를 뒤집기 or 앞선 전구 모두 뒤집지 않고, 현재 전구를 뒤집기
- 무조건 변해야하기 때문에 **증감량**의 최대를 구하자.
  
- dp[i] = i번째까지 최대 증감량 = max(dp[i-1] + 현재 전구 뒤집었을때 증감량, 현재 전구 뒤집었을때 증감량)

### 아이디어

- 최소 : dp
  - dp[i] = i개로 만들 수 있는 최소 숫자
    - i<=7까지 기저 조건 존재
  - minNum[8] = i개로 만들 수 있는 최소 숫자
  - dp[i] = min(dp[i], dp[i-j]*10 + minNum[j]);
  - 놓친 부분
    - i-j<2일 때, dp[i-j] = MAX이므로 자료형 오류 발생
- 최대 : 그리디

  - 짤수일 때, n/2만큼 1 출력

- 단순 dfs, 방문했던 정점에 또 방문하면 사이클이 생겼다고 판단하고 -1 출력 후 exit : 메모리 초과

- dp로 관리하되 이미 dp 값으로 기존에 방문한 적 있는 정점에 대해서는 다시 탐색하지 않고 기존 dp값 반환할 수 있도록 시간 관리

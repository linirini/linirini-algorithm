# [Gold V] 도로의 개수 - 1577 

[문제 링크](https://www.acmicpc.net/problem/1577) 

### 성능 요약

메모리: 14364 KB, 시간: 104 ms

### 분류

다이나믹 프로그래밍, 집합과 맵

### 아이디어

- dp[i][j] = dp[i-1][j] + dp[i][j-1]
- 공사 중인 도로 -> 계산에 미포함
  - 따라서, 공사 중인 도로에 해당한다면 dp[i][j]에서 상쇄
  - 세로 도로 공사: dp[i][j] -= dp[i][j-1]
  - 가로 도로 공사: dp[i][j] -= dp[i-1][j]

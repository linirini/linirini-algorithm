# [Gold V] 카드 바꾸기 - 25401 

[문제 링크](https://www.acmicpc.net/problem/25401) 

### 성능 요약

메모리: 2024 KB, 시간: 68 ms

### 분류

브루트포스 알고리즘, 구현, 수학

### 아이디어

- 바꿔야 하는 최대 카드 수 : n-2
  - 아무거나 장 집어서 등차수열 형태로 나머지 바꿔주면 됨
- 따라서 n^2 시간 복잡도로 답을 구할 수 있다.
- 2가지 카드를 뽑아(조합), 해당 구간 차 만큼 등차수열 형태로 바꿔야 할 카드의 개수를 구하고 최소를 갱신한다.

# [Gold V] 가장 긴 짝수 연속한 부분 수열 (large) - 22862 

[문제 링크](https://www.acmicpc.net/problem/22862) 

### 성능 요약

메모리: 5928 KB, 시간: 108 ms

### 분류

두 포인터

### 아이디어 

- S[i] : 홀수 개수 누적합 저장
- st, en 포인터 처음에서 시작해
  - 홀수 개수 <= K : (구간 길이 - 홀수 개수) 최대 갱신 및 en++
  - 홀수 개수 > K : st++
- 놓친 반례 : 최대 K개 원소 제거 가능 == 꼭 K개 제거 X

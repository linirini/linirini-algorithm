### 문제 설명

<p>3×N 크기의 벽을 2×1, 1×2 크기의 타일로 채우는 경우의 수를 구해보자.</p>

### 아이디어

- N이 홀수 : 방법 없음
- N이 짝수 :
 - N = 2 : 3 = dp[2]
 - N = 4 : dp[2] * dp[2] + 4일 때 고유 모양 2가지 = 11
 - N = 6 : dp[4] * dp[2] + 6일 때 고유 모양 2가지 = 35
 - N = k : dp[k-2] * dp[2] + dp[k-4] * 2 ... + 2

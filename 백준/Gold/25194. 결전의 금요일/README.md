### 아이디어

1) 백트래킹
- 금요일 : sum % 7 == 4
- 입력받은 값을 7로 나눈 나머지들의 조합 중에!
- 조합 -> 백트래킹!

2) dp
- dp[i]: i요일에 일을 끝낼 수 있는 여부
- 초기: dp[1(월)]=1
- 입력 받은 값에 대하여 `dp[i]=true인 요일` + `입력받은 값`의 dp값을 갱신
- dp[금]=true -> YES

# [Gold V] 두 개의 탑 - 2118 

[문제 링크](https://www.acmicpc.net/problem/2118) 

### 성능 요약

메모리: 2412 KB, 시간: 4 ms

### 분류

누적 합, 두 포인터

### 아이디어

- 거리 누적합 저장한 1차원 배열 활용
- 투 포인터로 처음에서 시작해서
- min((투 포인터 사이 구간 길이), (총 길이- 투포인터 사이 구간 길이)) 연산
- ans = max(ans, 위의 결과)
- 만약 투포인터 사이 구간 길이 < (총 길이- 투포인터 사이 구간 길이) : end 포인터 이동
- 그 외 : start 포인터 이동

### 아이디어

- N 제곱 형태를 구하므로 주어진 범위의 제곱근인 10^7까지의 소수만 구하면 된다.
- 에라토스네스의 체 사용
- 자료형 범위 주의
 - N제곱 고려 범위 : 현재 수*제곱수 < B
 - 하지만 현재수*제곱수 > long long 가능
 - 따라서 현재수 < B/i로 비교해야함

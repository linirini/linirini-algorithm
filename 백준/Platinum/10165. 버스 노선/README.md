### 아이디어

- 원형 -> 노선의 2배로 처리 : 예를 들어, 0~4번 노선이라면 원형임을 고려하여 0~8까지 고려
 - a < b -> (a,b)와 (a+N, b+N) 모두 노선으로 추가
 - a > b -> (a, b+N) 추가
- 노선 값 정렬해서 라인 스위핑 수행
- 다른 노선 안에 포함되는 노선에 대하여 방문 배열에 기록
